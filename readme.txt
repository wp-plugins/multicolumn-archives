Multicolumn archives widget.

== Description ==

This is a simple multicolumn archives widget.

Features:
* display months without posts
* display post counts
* custom column width
* custom column margins

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `multicolumn_archives.php` to the `/wp-content/plugins/` directory and unzip or use the internal plugin install
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the 'Multicolumn Archives' to the sidebar from the widget menu
